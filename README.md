# food-menu-website
🍔 Food Ordering Website

A simple and attractive food ordering website built using HTML and CSS. The project features a clean layout, food item cards, interactive order buttons, and modern styling using Flexbox.

Features
*Responsive layout structure
*Food menu cards
*Order buttons
*Hover effects
*Clean UI design
*Flexbox-based alignment

Technologies Used

*HTML5
*CSS3
*Flexbox
